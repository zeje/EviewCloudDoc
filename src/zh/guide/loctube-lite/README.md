## 介绍

loctube-lite提供webapi，可以快速解析eview tcp  protocol，将二进制解析成为json，与编程语言无关。
但是，您需要自己实现对设备的管理，也需要实现自己的业务类api接口。
与SDK相比，在设备量与消息量比较庞大的情况下，性能欠缺。
